# Storage Field

## dictadata field types

### Base Types

* "boolean"
* "integer"
* "number"
* "keyword"
* "text" | "string"
* "date"
* "uuid"
* "binary"

### Complex object types

* "list" - an array of values
* "map" - a nested object
* "geometry" - GeoJSON geometry object

List and map types may be nested.

### other types

* "unknown"  - used for values that are null
